##  Подробнее

*   **Умное авто-парри:**
*   **Аавто-клик (Спам):** Кликает со скоростью до 125 раз в секунду.
*   **Флик камеры (Camera Flick):** После удара камера резко дергается вбок, назад или вверх. Для кручёных мячей
*   **Авто-сохранение настроек (Нов):** Больше не надо каждый раз заново двигать ползунки и ставить бинды. Всё сохраняется в файл и само загружается при старте.
*   **Скрытие головы (Headless):** Делает голову невидимой (визуальный твик)
